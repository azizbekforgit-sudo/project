# План: Добавление полей email, город и смена пароля

## Задача
Добавить при регистрации поля email и город, показывать их в профиле с возможностью редактирования, добавить функцию смены пароля.

## Что нужно сделать

### 1. Backend (Python/FastAPI)

**models.py** — добавить колонку `city`:
```python
city = Column(String(100), nullable=True)
```

**schemas.py** — обновить схемы:
- `UserRegister`: email уже есть (Optional), добавить `city: Optional[str] = None`
- `ProfileUpdate`: добавить `city: Optional[str] = None`
- Новая схема `ChangePassword`: `current_password: str`, `new_password: str`

**routers/auth.py** — обновить эндпоинты:
- `POST /register`: добавить `city` при создании пользователя
- `PATCH /me`: добавить `city` при обновлении профиля
- Новый `POST /api/auth/change-password`: проверка текущего пароля + сохранение нового

**main.py** — добавить миграцию для новой колонки `city` (ALTER TABLE ADD COLUMN IF NOT EXISTS)

### 2. Frontend (Vanilla JS)

**pages/register.js** — добавить поля в форму:
- Email (input type="email", optional)
- Город (input text, optional)

**pages/profile.js** — обновить отображение:
- Показывать город в шапке профила
- Добавить город в секцию настроек (editable)
- Добавить секцию "Безопасность" с формой смены пароля

**api.js** — добавить метод:
```javascript
changePassword: (body) => request('POST', '/api/auth/change-password', { body })
```

- Зарегистрировать нового пользователя с email и городом
- Проверить что данные отображаются в профиле
- Отредактировать email и город в профиле
