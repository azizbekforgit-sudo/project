# План: Исправление системы доставки

## Проблема
Два отдельных набора данных (CourierOrder и DeliveryRequest) не связаны. Принятые DeliveryRequest не отображаются во вкладке "Принятые".

## Что нужно сделать

### Backend (delivery.py)
1. Добавить статусы: collecting, in_transit, delivered, completed
2. Добавить PATCH endpoint для смены статуса delivery_request
3. Добавить POST endpoint для оценки заказа покупателем (rating 0-10 + comment)
4. Добавить GET endpoint для завершённых заказов с рейтингами
5. Добавить поле `driver_earnings` в DeliveryRequest

### Frontend (delivery.js)
1. Исправить вкладку "Принятые" — загружать DeliveryRequest с driver_confirmed_disclaimer=true
2. Добавить модальное окно деталей заказа (откуда/куда, заработок, статус)
3. Добавить кнопки смены статуса: Собирается → В пути → Доставлено
4. Добавить экран подтверждения доставки покупателем (Да/Нет)
5. Добавить форму оценки (0-10 + комментарий)
6. Добавить вкладку "Выполненные" с рейтингами

### Frontend (profile.js)
1. Добавить секцию "Выполненные заказы" с рейтингами в профиль курьера
